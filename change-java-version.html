Change Java Version
Semisal kita punya banyak environment java dengan versi yang berbeda beda, tapi kita ingin switch environment entah downgrade atau upgrade begini caranya.
Cek versi java yang terinstal
/usr/libexec/java_home -V
Bash
Nanti keluar kayak gini
Ubah versi java
export JAVA_HOME=`/usr/libexec/java_home -v 1.8.0_292`
Bash
